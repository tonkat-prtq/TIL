#### カッコの省略

メソッド名の後に空白があるとメソッド呼び出しのカッコが省略されたと解釈される。

`foo (2) * 2`はメソッド名と引数の間に空白があるため、`foo((2) * 2)`が呼ばれたと解釈される。
よって、4の4乗の256になる。

```ruby
# 問題
# 次のコードを実行するとどうなりますか

def foo(n)
  n ** n
end

puts foo (2) * 2
# foo((2) * 2) と解釈される
```
